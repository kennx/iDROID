## 简介

Cap LoRa-1262 是一款高性能通信扩展模块，专为 **Cardputer-Adv** 设计，集成 **LoRa 无线通信** 和 **GNSS 全球定位** 功能。通过 HY2.0-4P 扩展接口与主控稳定连接。

---

## 核心特性

- 集成 LoRa 调制技术和 GNSS 全球定位系统
- 工作频率：**868 ~ 923 MHz**
- 支持调制模式：FSK、GFSK、MSK、GMSK、LoRa、OOK
- LoRa 接收灵敏度低至 **-147 dBm**（低速率模式）
- LoRa 最大发射功率：**+20 dBm**
- LoRa 外置 SMA 天线接口，GPS 内置陶瓷天线
- 定位支持多卫星导航系统（GPS / QZSS / BD2 / BD3 / GAL / GLO）
- 多频多系统接收
- HY2.0-4P 扩展接口
- 开发平台：Arduino、UiFlow2

| 参数 | 值 |
|------|------|
| LoRa 模块 | SX1262 |
| 频率范围 | 868 ~ 923 MHz |
| 可编程波特率 | 最高 300kbps |
| 通信接口 | SPI |

| 参数 | 值 |
|------|------|
| GPS 模块 | ATGM336H-6N @ AT6668 |

| 支持 GNSS 系统 | GPS / QZSS / BD2 / BD3 / GAL / GLO |
| 频段 | BDS: B1I+B1C；GPS/QZSS/SBAS: L1；GALILEO: E1；GLONASS: R1 |

| 定位精度 | <1.5m（CEP50） |

| 协议 | NMEA 0183 4.1 |
| 灵敏度 | 跟踪: -162dBm / 捕获: -160dBm / 冷启动: -148dBm |
